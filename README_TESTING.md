## 💡 أوامر Console المفيدة

### فحص المستخدم
```javascript
useAuthStore.getState().user?.email
// يجب أن يظهر البريد الإلكتروني
```

### فحص الملف الشخصي
```javascript
useAuthStore.getState().profile?.full_name
// يجب أن يظهر الاسم الكامل
```

### فحص المتجر
```javascript
useAuthStore.getState().store?.name
// يجب أن يظهر اسم المتجر
```

### فحص كل شيء مرة واحدة
```javascript
const check = () => {
  const state = useAuthStore.getState()
  console.log('✅ User:', state.user?.email)
  console.log('✅ Profile:', state.profile?.full_name)
  console.log('✅ Store:', state.store?.name)
  console.log('✅ Loading:', state.isLoading)
}
check()
```
