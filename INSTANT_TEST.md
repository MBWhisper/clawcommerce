```bash

```

## 🆘 إذا حدثت مشكلة

### خطأ: "Invalid login credentials"
```
✅ الحل: تأكد من إدخال البيانات الصحيحة
✅ الحل: استخدم نفس البريد الذي سجلت به
```

### خطأ: "Database error"
```
✅ الحل: تحقق من متغيرات البيئة في .env
✅ الحل: أعد تشغيل الخادم (Ctrl+C ثم npm run dev)
```

### لم يظهر الداشبورد
```
✅ الحل: افتح Console (F12) وابحث عن الأخطاء
✅ الحل: تحقق من الاتصال بـ Supabase
✅ الحل: أعد تحميل الصفحة (F5)
```

### الداشبورد يحمل بطيء
```
✅ الحل: انتظر قليلاً (أول مرة قد تكون أبطأ)
✅ الحل: افتح Console وتحقق من الـ Network tab
```
