## 🚀 خطوات الاختبار السريع (5 دقائق)

### الخطوة 1: تشغيل التطبيق
```bash
npm run dev
# أو
pnpm dev
```
التطبيق سيشتغل على http://localhost:5173

### الخطوة 2: الانتقال إلى صفحة التسجيل
```
رابط مباشر: http://localhost:5173/register
```

### الخطوة 3: ملء نموذج التسجيل (الحساب الأول)
```
الاسم الكامل: محمد علي
البريد الإلكتروني: test1@example.com
كلمة المرور: Test@123456
تأكيد كلمة المرور: Test@123456

اضغط: زر التسجيل
```

### الخطوة 4: التحقق من النجاح
✅ يجب أن تنتقل إلى الداشبورد مباشرة
✅ يجب أن ترى رسالة ترحيب باسمك
✅ يجب أن ترى بيانات متعددة (الإحصائيات والطلبات والمنتجات)

### الخطوة 5: اختبار تسجيل الدخول
```
1. اضغط على أيقونة المستخدم في الداشبورد
2. اضغط على "تسجيل الخروج"
3. انتقل إلى صفحة تسجيل الدخول: http://localhost:5173/login
4. أدخل بيانات الحساب نفسه:
   - البريد الإلكتروني: test1@example.com
   - كلمة المرور: Test@123456
5. اضغط: زر تسجيل الدخول
```

✅ يجب أن تنتقل إلى الداشبورد مرة أخرى

## 🔍 استكشاف الأخطاء المتقدم

### استخدام Console (F12)
```javascript
// فحص جودة الاتصال
console.log('[v0] Testing Supabase connection');

// عرض بيانات المستخدم الحالي
const { data: { user } } = await supabase.auth.getUser();
console.log('Current user:', user);

// عرض ملف تعريف المستخدم
const { data: profile } = await supabase
  .from('profiles')
  .select('*')
  .eq('id', user.id)
  .single();
console.log('User profile:', profile);

// عرض المتجر الخاص بالمستخدم
const { data: store } = await supabase
  .from('stores')
  .select('*')
  .eq('user_id', user.id)
  .single();
console.log('User store:', store);
```

## ⚠️ رسائل الخطأ الشائعة وحلولها

### خطأ: "Invalid login credentials"
**السبب:** بيانات الدخول غير صحيحة
**الحل:** تأكد من البريد الإلكتروني وكلمة المرور

### خطأ: "Database error saving new user"
**السبب:** مشكلة في الاتصال بقاعدة البيانات
**الحل:** تحقق من متغيرات البيئة (VITE_SUPABASE_URL و VITE_SUPABASE_ANON_KEY)

### خطأ: "User already registered"
**السبب:** البريد الإلكتروني مسجل بالفعل
**الحل:** استخدم بريد إلكتروني مختلف

### خطأ: "Error sending confirmation email"
**السبب:** مشكلة في إعدادات البريد في Supabase
**الحل:** اطلع على EMAIL_SETUP_GUIDE.md
