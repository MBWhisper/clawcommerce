# اختبار البدء السريع ⚡

**الوقت المتوقع:** 5 دقائق  
**المتطلبات:** متصفح + اتصال إنترنت

---

## الخطوة 1️⃣ - التسجيل

### 1. اذهب إلى صفحة التسجيل
```
URL: /register
أو: https://your-domain.com/register
```

### 2. ملأ النموذج
```
الاسم الكامل:     أحمد محمد
البريد الإلكتروني: ahmed@test.com
كلمة المرور:     Test@12345
تأكيد المرور:    Test@12345
```

### 3. اضغط الزر
```
"إنشاء حساب مجاني"
```

### ✅ النتيجة المتوقعة
- [ ] لا توجد رسالة خطأ
- [ ] الصفحة تحمل
- [ ] الانتقال للداشبورد
- [ ] اسمك يظهر: "مرحباً، أحمد محمد"

---

## الخطوة 2️⃣ - الداشبورد

### 1. تحقق من البيانات

```javascript
// في Console (F12):
useAuthStore.getState().user?.email
// يجب أن يظهر: ahmed@test.com

useAuthStore.getState().profile?.full_name
// يجب أن يظهر: أحمد محمد

useAuthStore.getState().store?.name
// يجب أن يظهر: أحمد محمد's Store
```

### 2. تصفح الداشبورد
- [ ] القائمة الجانبية مرئية
- [ ] الشريط العلوي مرئي
- [ ] الإحصائيات تحمل
- [ ] الأزرار تعمل

### ✅ النتيجة المتوقعة
- [ ] جميع الأقسام تحمل
- [ ] الإحصائيات تظهر (0 حالياً)
- [ ] أي رسائل خطأ؟ ❌ يجب ألا تكون

```

### 3. أدخل بيانات تسجيل الدخول
```

### 4. اضغط الزر
```

```

### ✅ النتيجة المتوقعة

- [ ] اسمك يظهر مرة أخرى

## إذا واجهت مشكلة

### 1. افتح Console
```
F12 أو Ctrl+Shift+I أو Right Click > Inspect
```

### 2. اقرأ رسالة الخطأ
```
ابحث عن أي رسائل حمراء
```

### 3. شيء مشترك:
```
❌ "Cannot find module" 
   → انتظر التحميل الكامل

❌ "Invalid credentials"
   → تأكد من Supabase env

❌ "Database error"
   → قد تحتاج لإنشاء جداول
```

## نصائح

💡 **استخدم Console للتحقق السريع:**
```javascript
// اختبر كل شيء مرة واحدة
const check = () => {
  console.log('User:', useAuthStore.getState().user?.email)
  console.log('Store:', useAuthStore.getState().store?.name)
  console.log('Profile:', useAuthStore.getState().profile?.full_name)
}
check()
```

💡 **امسح البيانات إذا لزم الأمر:**
```javascript
localStorage.removeItem('auth-storage')
location.reload()
```

## النتيجة النهائية

✅ **إذا مرّ كل الاختبارات:**
- النظام **جاهز للاستخدام**
- جميع الميزات **تعمل بشكل صحيح**
- يمكنك **دعوة المستخدمين**

❌ **إذا فشل أي اختبار:**
- راجع أوامر Console أعلاه
- افتح ملف `BROWSER_TESTING_COMMANDS.md`
- تحقق من Supabase env variables
