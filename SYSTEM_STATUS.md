## 🐛 حل المشاكل الشائعة

### مشكلة: "فشل إنشاء الحساب"
**الحل:**
```
1. تحقق من Supabase env variables
2. اذهب إلى Settings > Vars
3. تأكد من VITE_SUPABASE_URL و VITE_SUPABASE_ANON_KEY
```

### مشكلة: "البريد الإلكتروني مسجل بالفعل"
**الحل:**
```
هذا طبيعي - البريد موجود بالفعل
حاول بريد جديد أو استخدم "نسيت كلمة المرور"
```

### مشكلة: "خطأ في إرسال البريد"
**الحل:**
```
1. اتبع: /setup/email
2. قم بإعداد مزود بريد SMTP
3. أو عطّل تأكيد البريد للتطوير
```

### مشكلة: "الداشبورد فارغ"
**الحل:**
```
1. حدّث الصفحة (Ctrl+R)
2. افحص Console (F12)
3. تأكد من وجود متجر في قاعدة البيانات
```
